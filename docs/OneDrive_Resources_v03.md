# StudyHub OneDrive Resources v0.3

This version adds OneDrive-linked resources.

## What works now

- Add a resource
- Store title, type, subject, author, edition/week
- Store file name and file size
- Store OneDrive link
- Store local/OneDrive folder location
- Store index status: Not indexed, Ready to index, Indexed
- Store tags
- Store chapter/section list
- Open resource detail page
- Open OneDrive file from StudyHub
- Search resource metadata
- Filter by resource type
- Data saves in browser localStorage for now

## What to do first

1. Create a OneDrive folder named `StudyHub Resources`
2. Add folders:
   - `01 Textbooks`
   - `02 Lecture Slides`
   - `03 Kaplan NCLEX`
   - `04 Professor Handouts`
   - `05 Diagrams Images`
   - `06 Remediation`
3. Upload one test file
4. Copy its OneDrive share link
5. Add it in StudyHub Library
